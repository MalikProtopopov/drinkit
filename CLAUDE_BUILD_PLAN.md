# CLAUDE BUILD PLAN — Juicy V2

Источник: `Juicy_документы/Juicy_USER_STORIES_по_ролям.md` (32 user stories) + смета (блоки ядра/опций).
Стек: FastAPI + SQLAlchemy (PostgreSQL prod / SQLite dev+tests) + Redis (опционально, in-memory fallback) · Next.js (прототип `app/` дорабатывается и подключается к API).

Фичи сгруппированы по блокам сметы (зависимости — сверху вниз).

## Git-процесс
После завершения каждой фичи (тесты зелёные):
1. `git add -A`
2. `git commit -m "feat(FXX): <краткое описание фичи>"`
3. `git push origin main` — репозиторий: https://github.com/MalikProtopopov/drinkit.git
4. Отметить фичу выполненной в разделе Progress (в том же коммите).
